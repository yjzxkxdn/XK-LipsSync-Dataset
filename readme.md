# XK口型数据集

### 数据概述
本数据集总时长约35分钟，在没有声学装修的安静房间录制，使用 **LipsSync** 获取口型数据。音频未经过处理，在实际使用时可根据需求进行对齐或降噪操作。

### 录制环境与设备信息
- **录制环境**：安静的房间（无声学装修）
- **录制设备**：
  - iPhone X
  - 红米 Note 11T Pro
  - SM57 动圈麦克风
  - 杂牌电容麦克风

> **注意**：由于 iPhone X 性能限制，部分较长的歌曲需分多次录制完成。
